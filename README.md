# ChatTale - Chat Story Maker

Create fake text message conversations and export them as videos or screenshots for TikTok, Instagram, and YouTube.

## Features

### Create Realistic Chat Conversations

**Multiple Themes**
- iMessage (iOS style)
- WhatsApp
- Facebook Messenger
- Discord

**Characters**
- Create unlimited characters
- Custom names and colors
- Avatar options: emoji or photo upload
- Set as sender (right) or receiver (left)

**Messages**
- Text messages
- Image messages (photo picker)
- Drag-to-reorder messages
- Edit or delete any message

### Export Options

**Video Export**
- Typing animation with cursor
- Customizable typing speed (Slow/Normal/Fast)
- Optional keyboard display
- Optional typing indicator (...)
- Sound effects option

**Screenshot Export**
- High-quality static image
- Include/exclude avatars
- Include/exclude timestamps
- Include/exclude reactions

**Formats**
- TikTok (9:16 vertical)
- Instagram (1:1 square)
- YouTube (16:9 horizontal)

**Dark Mode**
- Export with dark background

### Organization

**Folders**
- Create colored folders
- Organize conversations by category
- Move conversations between folders

**Search**
- Search by conversation title
- Search within message content

**Duplicate**
- Copy conversations as templates
- Deep copies all messages, characters, settings

## How to Use

1. **Create a Conversation**
   - Tap + to create new
   - Enter title and choose theme

2. **Add Characters**
   - Long-press character button to edit
   - Set name, color, avatar
   - Choose sender (right) or receiver (left)

3. **Write Messages**
   - Select a character
   - Type message or tap photo icon
   - Messages appear in chat preview

4. **Add Details**
   - Long-press message for options
   - Add reactions, timestamps, delivery status

5. **Reorder Messages**
   - Tap arrow icon in toolbar
   - Drag to reorder
   - Swipe to delete

6. **Export**
   - Tap share icon
   - Choose Video or Screenshot
   - Select format and settings
   - Export and share!

## Tips

- Use **timestamps** strategically to show time passing
- Add **reactions** for emotional moments
- Use **delivery status** to build tension (message left on "Read")
- **Duplicate** conversations to create variations
- Organize with **folders** for different content series
